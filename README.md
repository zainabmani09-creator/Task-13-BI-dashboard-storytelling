# Retail Sales KPI Dashboard (Power BI)

This project contains a KPI dashboard built using Power BI for Task 13 – BI Dashboard Storytelling.

## Files
- dashboard.pbix → Power BI source file
- dashboard_export.pdf → Exported dashboard
- insights_task13.txt → Key business insights

## KPIs Included
- Total Sales
- Total Quantity
- Average Order Value

## Visuals
- Sales trend over time
- Sales by product category
- Sales by gender
- Top 10 customers
- Slicers for Date, Gender, Category
